ccg-router

ANTHROPIC_BASE_URL local router

Use ANTHROPIC_BASE_URL to point Claude Code at ccg-router, a local Anthropic-compatible proxy for routing and usage tracking.

export ANTHROPIC_BASE_URL=http://127.0.0.1:17180
ccg-router start

ccg-router keeps provider keys in local environment variables or local config and records request metadata in a local SQLite ledger.
