ccg-router

Codex CLI OpenAI-compatible local router

ccg-router exposes an OpenAI-compatible local endpoint for Codex CLI and can share the same daemon with Claude Code traffic.

Codex CLI endpoint

export OPENAI_BASE_URL=http://127.0.0.1:17180
ccg-router start

What it gives you

  • One local OpenAI-compatible proxy for AI coding CLI traffic.
  • Shared upstream config with Anthropic-compatible routes.
  • Local usage metadata in SQLite.
