ccg-router

Claude Code router

ccg-router exposes an Anthropic-compatible local endpoint for Claude Code. It lets you route Claude Code traffic through upstreams you configure while keeping provider keys local.

Set Claude Code endpoint

export ANTHROPIC_BASE_URL=http://127.0.0.1:17180
ccg-router start

Use cases

  • Route Claude Code to an Anthropic-compatible API.
  • Track Claude Code request metadata locally.
  • Share one routing config with Codex CLI.
